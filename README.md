# astrbot_plugin_dnd_dice

适用于 [AstrBot](https://github.com/AstrBotDevs/AstrBot) 的 DnD D20 掷骰插件，支持完整骰池语法、技能检定判定，以及 LLM 函数工具自动调用。

## 功能

- 基础骰池：`d20`、`2d6+5`、`4d6kh3` 等完整 DnD 语法
- 优势 / 劣势骰：`d20adv` / `d20dis`
- 爆炸骰：`d6!`
- 多骰组合：`2d6+1d4+3`
- 标签注释：`d20 攻击检定`、`d20#攻击检定`
- 技能检定 DC 判定：`d20 感知 15` → 输出成功 / 失败
- 天然 20 / 天然 1 特殊提示：大成功 / 大失败
- LLM 函数工具：LLM 可在 TRPG 叙事中自动调用掷骰

## 安装

在 AstrBot 管理面板的插件市场中搜索 `astrbot_plugin_dnd_dice` 并安装，或将本仓库克隆至 `data/plugins/` 目录下。

## 指令

| 指令 | 说明 |
|------|------|
| `/r` | 默认掷一个 d20 |
| `/r [表达式]` | 按指定骰池表达式掷骰 |
| `/roll [表达式]` | `/r` 的别名 |

## 语法示例

| 输入 | 输出示例 |
|------|----------|
| `/r d20` | `d20: [14] = 14` |
| `/r 1d20+5` | `d20+5: [9] +5 = 14` |
| `/r 2d6-1` | `2d6-1: [4, 3] -1 = 6` |
| `/r 4d6kh3` | `4d6kh3: [(2), 4, 5, 6] = 15` |
| `/r d20adv` | `d20adv: [(8), 17] = 17` |
| `/r d20dis` | `d20dis: [11, (15)] = 11` |
| `/r d6!` | `d6!: [6!, 4] = 10` |
| `/r 2d6+1d4+3 伤害` | `伤害 2d6+1d4+3: [3, 5] [2] +3 = 13` |
| `/r 1d20+5#攻击检定` | `攻击检定 d20+5: [13] +5 = 18` |
| `/r d20 感知 15` | `感知 d20: [12] = 12 / 15 失败` |
| `/r d20感知15` | `感知 d20: [18] = 18 / 15 成功` |
| `/r d20+3 奥秘 12` | `奥秘 d20+3: [10] +3 = 13 / 12 成功` |
| `/r d20adv 察觉 13` | `察觉 d20adv: [(7), 16] = 16 / 13 成功` |

### DC 技能检定规则

在表达式后附加 `标签 DC值`（有无空格均可），插件将自动比较掷骰总计与 DC：

- 总计 >= DC → 成功
- 总计 < DC → 失败
- 天然 20 → 大成功（无论 DC）
- 天然 1 → 大失败（无论 DC）

## 配置项

在 AstrBot 管理面板的插件配置中可调整以下参数：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `max_dice_count` | 整数 | 100 | 单次投掷最大骰子数量 |
| `max_dice_sides` | 整数 | 1000 | 骰子最大面数 |
| `exploding_max_depth` | 整数 | 20 | 爆炸骰最大递归深度 |
| `show_detail` | 布尔 | true | 是否显示各骰子点数明细 |

## LLM 函数工具

插件注册了 `roll_dice` 工具，启用 Agent 模式后，LLM 可在 TRPG 叙事中主动调用掷骰，结果将同时发送给用户并返回给 LLM 用于继续叙事。

LLM 调用参数：

| 参数 | 类型 | 说明 |
|------|------|------|
| `expression` | string | 骰池表达式，如 `"1d20+5"`、`"4d6kh3"` |
| `label` | string | 投掷说明，可附 DC 值，如 `"感知 15"`、`"感知15"` |
